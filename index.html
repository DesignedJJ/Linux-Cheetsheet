Linux Cheatsheet

Click a command card to expand for deeper explanations • Colors: White / Navy / Orange • Font: Poppins

pwd Print current directory Example: pwd ▶
Shows your current working directory (absolute path). Use before file ops to avoid mistakes.
ls List directory contents Example: ls -la ▶
Lists files/dirs. -l long format, -a include hidden, -h human sizes (with -l).
cd Change directory Example: cd /var/log ▶
Move around the filesystem. cd .. up a level, cd ~ to home, cd - to previous dir.
mkdir Create directory Example: mkdir -p projects/app/logs ▶
Creates directories. -p makes parent dirs as needed for nested paths.
rmdir Remove empty directory Example: rmdir old_folder ▶
Deletes an empty directory. For non-empty, use rm -r (careful).
rm Delete files/directories Example: rm -rf folder_name ▶
Deletes files/dirs. -r recursive, -f force (no prompt). Irreversible; double check path.
touch Create empty file Example: touch file.txt ▶
Creates an empty file or updates its modified time if it exists.
cp Copy file/folder Example: cp -r src/ dst/ ▶
Copies files. -r for dirs, -i prompt before overwrite, -a preserve attributes.
mv Move/rename Example: mv file.txt newfile.txt ▶
Moves or renames files/dirs. Also used to organize into directories.
cat View file content Example: cat file.txt ▶
Prints contents to terminal. Best for small files or piping to other tools.
nano Edit file (Nano) Example: nano config.cfg ▶
Beginner-friendly editor. Save: Ctrl+O, Exit: Ctrl+X, Search: Ctrl+W.
vim Edit file (Vim) Example: vim index.html ▶
Modal editor. i insert, Esc :wq save & quit, :q! quit without saving.
head First lines of file Example: head -n 5 file.txt ▶
Peek at headers. Combine with pipes to inspect data quickly.
tail Last lines of file Example: tail -n 5 log.txt ▶
See recent lines. Use -f to follow a growing file (logs).
grep Search in text Example: grep -i "error" /var/log/syslog ▶
Matches patterns. -i case-insensitive, -n line numbers, -C NUM context, -r recursive.
find Locate files Example: find /etc -name "*.conf" ▶
Search by name, type (-type f/d), size (-size), time (-mtime). Use -exec for actions.
locate Quick path lookup Example: locate .bashrc ▶
Searches an index (updatedb). Very fast for known paths.
chmod Change permissions Example: chmod 755 script.sh ▶
Set permissions (numeric/symbolic). 7=rwx,6=rw-,5=r-x,4=r--. Dirs need x to enter.
chown Change owner/group Example: sudo chown user:group file.txt ▶
Change file owner/group. -R for recursive. chgrp changes only group.
df Disk space usage Example: df -h ▶
Filesystem usage across mounts. -h human readable.
du Folder/file size Example: du -sh folder/ ▶
Summarize usage. -s summary only, -h human readable. Great for finding large dirs.
free Memory usage Example: free -h ▶
RAM and swap stats. For live detail, use top/htop.
top Process viewer Example: top ▶
Interactive process list. Sort by CPU; M sorts by memory; k to kill by PID.
htop Enhanced process viewer Example: htop ▶
Colorized tree view (if installed), mouse support—easier than top.
ps List processes (snapshot) Example: ps aux | grep nginx ▶
One-shot view. Combine with grep to filter.
kill Terminate process Example: kill -15 1234 ▶
Send signal to PID. 15=TERM graceful; 9=KILL force (last resort).
killall Kill by name Example: killall firefox ▶
Terminates all processes with a given name (use carefully).
man Manual pages Example: man tar ▶
Open command docs. / to search, n for next, q to quit.
history Command history Example: history | tail ▶
Review previous commands. !N reruns entry N; history -c clears.
tar Archive/compress Example: tar -czvf archive.tar.gz folder/ ▶
c create, x extract, z gzip, v verbose, f file. Extract with -xzvf -C dest/.
unzip Extract ZIP Example: unzip archive.zip -d out/ ▶
List with -l; choose destination with -d.
wget Download from URL Example: wget -O file.txt https://example.com/file.txt ▶
Non-interactive fetch. Resume with -c; use for simple downloads.
curl Transfer data Example: curl -I https://api.example.com ▶
Versatile: headers (-I), output (-o), data (-d), custom header (-H).
ssh Secure remote login Example: ssh -i ~/.ssh/key user@10.0.0.1 ▶
Encrypted shell to remote hosts. Config: ~/.ssh/config.
scp Secure file transfer Example: scp -r dir user@host:/path ▶
Copy over SSH. -r for dirs; -i specify key.
sudo Run as superuser Example: sudo apt update ▶
Execute with elevated privileges (sudoers needed).
apt Debian/Ubuntu packages Example: sudo apt install nginx ▶
Install/update/remove packages. Run apt update; apt upgrade regularly.
yum RHEL/CentOS packages Example: sudo yum install httpd ▶
Older RHEL/CentOS. On newer, use dnf (same syntax).
reboot Reboot system Example: sudo reboot ▶
Safely restart the machine.
shutdown Shutdown system Example: sudo shutdown now ▶
Power off. Use shutdown -r now to reboot; schedule with a time.
Service Management (systemd vs SysV) Tap to expand ▶ 
Modern (systemd):
• systemctl start  — start now
• systemctl stop  — stop
• systemctl restart  — restart (apply changes)
• systemctl status  — status + recent logs
• systemctl enable  — start at boot
• systemctl disable  — disable at boot
• journalctl -u  — view service logs

Legacy/compat (SysV init):
• service  start|stop|restart|status
OS / Kernel Information Tap to expand ▶ 
• cat /etc/os-release — distro name/version
• lsb_release -a — LSB info (if installed)
• uname -a — kernel version/arch
Viewing Logs & Common Log Files Tap to expand ▶ 
Viewing:
• less /var/log/syslog — pager with search (/)
• tail -f /var/log/syslog — follow live logs
• grep -i 'error' /var/log/syslog — search case-insensitive
• journalctl -xe — recent systemd logs with context

Common logs under /var/log/:
• /var/log/syslog or /var/log/messages — general messages
• /var/log/auth.log — authentication/sudo activity
• /var/log/kern.log — kernel messages
• /var/log/dpkg.log — package history (Debian/Ubuntu)
• App logs e.g., /var/log/nginx/, /var/log/apache2/
(Many require sudo to read.)
Users, Groups, Permissions — Quick Ref Tap to expand ▶ 
Users & groups:
• useradd / usermod / userdel — manage users
• groupadd / groupmod / groupdel — manage groups
• id  — show UID/GID and group memberships

Ownership & permissions:
• chown user:group path — change owner & group
• chgrp group path — change group only
• chmod 644 file — rw-r--r-- (owner read/write; others read-only)
  Octal: 7=rwx, 6=rw-, 5=r-x, 4=r--, 0=---
  Symbolic: chmod u=rwx,g=rx,o=r file
Notes: Directories need 'x' to enter. Use -R for recursive.
